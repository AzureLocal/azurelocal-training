# Module 00: Introduction to Azure Local

**Level:** L100 | **Duration:** 2 hours | **Prerequisites:** None | **Hands-on:** Presentation + optional demo

!!! note "Module Status"
    Framework module — content in development.

## Learning Objectives

- Explain what Azure Local is and how it differs from Windows Server and the legacy Azure Stack HCI
- Describe the Azure Local architecture at a high level (cloud-connected, Arc-managed, always-current)
- Identify common use cases — edge, branch office, VDI, AKS, sovereignty / regulated workloads
- Understand the hardware certification model (Validated, Integrated, Premier)
- Understand the licensing and subscription model

## Topics

1. What Is Azure Local? — successor to Azure Stack HCI, cloud-first hybrid platform
2. Azure Local vs. Windows Server vs. legacy Azure Stack HCI
3. Architecture Overview — Arc fabric, on-premises infrastructure, the "always-current" model
4. Hardware Certification Tiers — Validated, Integrated, Premier
5. Common Use Cases — edge / branch office / VDI / AKS / regulated / sovereignty
6. Licensing and Subscription Model
7. Planning an Azure Local Project — what's next

## Hands-on

Presentation + optional live tour of an existing Azure Local cluster in the Azure Portal. No lab.
