# Module 03: Management

**Level:** L200 | **Duration:** 3 hours | **Prerequisites:** Module 02 | **Hands-on:** Lab

!!! note "Module Status"
    Framework module — content in development.

## Learning Objectives

- Manage Azure Local clusters through the Azure Portal as the primary management plane
- Use PowerShell and Azure CLI for common management tasks
- Use Windows Admin Center (WAC) in its situational role — operator fallback and Dell OMSWAC integration
- Understand identity and RBAC for Azure Local
- Distinguish workload management from infrastructure management

## Topics

1. Azure Portal — Primary Management Plane — cluster overview, node management, RBAC
2. PowerShell & Azure CLI — Common operator commands
3. Windows Admin Center — When to use (operator fallback, no Portal/CLI access)
4. Dell OMSWAC Extension — Hardware integration, HCI Insights, AX node features
5. Identity & RBAC — Entra ID, role assignments, least-privilege
6. Workload vs. Infrastructure Management — framing and separation of concerns

## Hands-on

**Lab:** Tour the Portal — cluster blade, nodes, resources. Run common PowerShell/CLI commands. Connect WAC + Dell OMSWAC (where AX hardware is available) and review HCI Insights.
