# Module 11: Observability & Monitoring

**Level:** L200–L300 | **Duration:** 3 hours | **Prerequisites:** Module 03 | **Hands-on:** Lab

!!! note "Module Status"
    Framework module — content in development.

## Learning Objectives

- Deploy and configure the Azure Monitor Agent on Azure Local nodes
- Use Monitor Insights for Azure Local to visualize cluster health
- Configure Log Analytics workspaces and queries
- Build operational dashboards
- Configure alert rules and action groups
- Use HCI Insights via WAC/OMSWAC for hardware health

## Topics

1. Azure Monitor Architecture for Azure Local
2. Azure Monitor Agent — deployment, data collection rules
3. **Monitor Insights for Azure Local** — out-of-box dashboards
4. Log Analytics — workspace design, KQL queries
5. Operational Dashboards
6. Alert Rules and Action Groups
7. **HCI Insights** via WAC/OMSWAC — hardware health (for Dell AX in particular)
8. Performance Baselining

## Hands-on

**Lab:** Connect cluster to a Log Analytics workspace. Enable Monitor Insights. Build a dashboard. Configure an alert rule with email action group. View HCI Insights via WAC.
