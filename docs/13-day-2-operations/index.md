# Module 13: Day-2 Operations & Lifecycle

**Level:** L300 | **Duration:** 4 hours | **Prerequisites:** Modules 05–07 | **Hands-on:** Lab

!!! note "Module Status"
    Framework module — content in development.

## Learning Objectives

- Use Azure Update Manager to assess, schedule, and apply updates
- Perform Lifecycle Manager (LCM) update operations
- Expand cluster capacity and add nodes
- Plan and execute upgrades
- Manage the Azure Local roadmap and version compatibility

## Topics

1. **Azure Update Manager** — assessment, compliance, scheduling
2. **Lifecycle Manager (LCM)** — Azure Local update orchestration
3. Solution Updates — Microsoft + hardware partner integration
4. Capacity Expansion — adding storage, memory, network capacity
5. Node Additions — adding a new node to an existing cluster
6. Upgrades — version-to-version upgrade paths
7. Roadmap Planning — staying current

## Hands-on

**Lab:** Run an update assessment via Azure Update Manager. Apply a non-disruptive update via LCM. Walk through node addition (demo if no spare hardware).
