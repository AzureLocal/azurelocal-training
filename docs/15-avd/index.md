# Module 15: Azure Virtual Desktop on Azure Local

**Level:** L300 | **Duration:** 4 hours | **Prerequisites:** Module 05 | **Hands-on:** Lab

!!! note "Module Status"
    Framework module — content in development.

## Learning Objectives

- Describe AVD on Azure Local architecture and benefits (data locality, compliance, latency)
- Deploy an AVD host pool on Azure Local
- Deploy session hosts via Arc VMs
- Configure FSLogix for user profile management
- Implement scaling plans and autoscale

## Topics

1. AVD on Azure Local Architecture
2. Benefits — Data Locality, Compliance, Low Latency
3. Host Pool Deployment on Azure Local
4. Session Host Deployment — Arc VMs as session hosts
5. Image Management — base images, custom images
6. FSLogix Profile Containers
7. Scaling Plans and Autoscale
8. Monitoring AVD Session Health

## Hands-on

**Lab:** Deploy an AVD host pool on Azure Local. Add session hosts. Configure FSLogix. Connect a test user. Configure autoscale.
