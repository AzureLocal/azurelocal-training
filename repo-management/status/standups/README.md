# Standups

Running log of working session notes — decisions made, blockers hit, and items to carry forward.

## Format

Each file covers one session or sprint. Name files `YYYY-MM-DD.md` or `YYYY-MM-DD-topic.md`.

```
# YYYY-MM-DD

## Done
- ...

## Decisions
- ...

## Blockers
- ...

## Next
- ...
```

Files accumulate here over time. The most recent file is the most relevant.
